# Los Jojaras – GitHub Pages

Det här är en ren statisk hemsida för GitHub Pages.

## Viktigt

Ladda upp alla filer och mappar exakt som de ligger här:

```text
index.html
style.css
script.js
.nojekyll
images/
  live-poster.jpg
  logo-wide.jpg
  bomb-logo.jpg
  cartoon-live.jpg
```

Alla bildsökvägar är relativa och börjar med `./images/`, vilket fungerar bra för GitHub Pages även när sidan ligger under `https://användare.github.io/repository/`.

## Publicera på GitHub Pages

1. Skapa ett nytt repository på GitHub.
2. Ladda upp hela innehållet i den här mappen till repositoryts root.
3. Gå till **Settings → Pages**.
4. Välj **Deploy from a branch**.
5. Välj branch **main** och folder **/root**.
6. Spara och vänta någon minut.

## Ändra innehåll

- Text och länkar: `index.html`
- Design: `style.css`
- Meny, animationer och bildvisning: `script.js`
- Bilder: `images/`
